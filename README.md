## ✨ Features

- ⚡ Real-time device monitoring using WebSockets (no refresh required)  
- 🔗 Universal device connectivity via MQTT protocol  
- 📊 Interactive dashboard with live charts, gauges, and controls  
- 🎛️ Remote device control (ON/OFF commands with instant response)  
- 🤖 AI-based anomaly detection for smart alerts  
- 📲 Instant notifications via Telegram & SMS integration  
- 👥 Role-Based Access Control (RBAC) for secure multi-user usage  
- 🔐 Secure authentication using JWT and password encryption (Bcrypt)  
- ☁️ Long-term data storage (1+ year history for analytics)  
- 🔄 Lightweight OTA updates for remote firmware management  
- 💻 Web REPL console for executing MicroPython code remotely

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/techcoderRohit/Sentinel-Cloud.git
cd Sentinel-Cloud

# Backend setup
cd backend
npm install
npm run dev

# Frontend setup (open new terminal)
cd frontend
npm install
npm start
```
